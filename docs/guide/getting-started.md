# Getting Started

## Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| Node.js | 20+ | ESM modules required |
| OpenCode CLI | latest | Must be on `PATH` |
| Telegram Bot Token | — | From [@BotFather](https://t.me/BotFather) |

## Installation

```bash
git clone https://github.com/wngfra/Nuntia.git
cd Nuntia
npm install
```

## Configuration

Copy the example environment file and fill in the required values:

```bash
cp .env.example .env
```

Three variables are required to start:

```dotenv
# Get from @BotFather on Telegram
TELEGRAM_BOT_TOKEN=123456:ABC-DEF1234...

# Your Telegram chat ID (or group ID). Comma-separated for multiple.
ALLOWED_CHAT_IDS=123456789

# Generate with: openssl rand -hex 32
MASTER_KEY=a1b2c3d4...
```

::: tip Finding your chat ID
Send any message to [@userinfobot](https://t.me/userinfobot) on Telegram to get your numeric chat ID.
:::

See [Configuration](/guide/configuration) for the full list of environment variables.

## Running

### Development

```bash
npm run dev
```

Uses `tsx` for on-the-fly TypeScript execution. The bot starts immediately — no build step needed.

### Production

```bash
npm run build
npm start
```

Compiles TypeScript to `dist/`, then runs the compiled JavaScript.

### Docker

```bash
cp .env.example .env
# Edit .env with your values
docker compose up -d
```

## First Interaction

Once the bot is running, open Telegram and send `/start` to your bot. You'll see:

```
Welcome to Nuntia!

Active project: default
Model: claude-sonnet-4-20250514

Send any message to start coding, or use /help for commands.
```

Now send a natural language coding request:

```
Build a simple Express.js hello world server with a health check endpoint
```

The bot will:

1. Prepend the TDD workflow system prompt
2. Spawn OpenCode in your project directory
3. Stream agent stage transitions to chat in real-time
4. Auto-commit the result to a new Git branch
5. Send a summary with files changed and branch name

## What Happens Under the Hood

```
You (Telegram) → Nuntia Bot → OpenCode CLI → Agent Pipeline
                                                     │
                  ← Streaming output (stage-by-stage) ←
                  ← Task summary + Git branch name    ←
```

Each chat maps to an isolated project directory:

```
workspaces/
└── tg_123456789/
    └── default/
        ├── uploads/
        ├── .git/
        ├── SPEC.md        ← Generated by OpenCode
        ├── tests/          ← TDD tests
        ├── src/            ← Implementation
        ├── README.md
        └── CHANGELOG.md
```
